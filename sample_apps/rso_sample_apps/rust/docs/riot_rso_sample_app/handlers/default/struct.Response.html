pub struct Response {
    sign_in_url: String,
}
Expand description

Represents the server’s response to a request with a sign-in URL.

This struct is used to generate the HTML response for the client, directing them to the sign-in page. It leverages the Askama template engine to render the default.html template with the provided sign_in_url.
